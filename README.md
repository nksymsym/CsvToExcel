# CsvToExcel

csvをExcelに貼り付けて表をいい感じにするやつ

## いる

.NET Framework 4.7.2

## 使い方

set pg="exeのパス"
set csvDir="csvファイルのあるディレクトリ"
set excelDir="excelファイルを出力するディレクトリ"
set nameBase="excelファイルのベース部分（※後ろにタイムスタンプと拡張子が付きます）"
%pg% %csvDir% %excelDir% %nameBase%
